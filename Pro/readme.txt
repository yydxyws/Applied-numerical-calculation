这里面是热对流扩散方程的物理模型和解析解，直接复制到 latex 编译即可。
下面四个程序用到的都是这个解析解提到的 Example 部分给出的条件。

-stable_FEM
这是稳态热对流方程（不含时）有限差分法求解的程序，直接运行即可，需要截图的话直接在运行得到的结果截图

-stable_FEM
这是稳态热对流方程（不含时）有限元法求解的程序，直接运行即可，需要截图的话直接在运行得到的结果截图。

-Unstable_FEM
这是非稳态热对流方程（含时）有限差分法求解的程序，直接运行即可（得到的是动态图），需要截图的话直接在运行得到的结果截图

-Unstable_FEM
这是稳态热对流方程（含时）有限元法求解的程序，直接运行即可（得到的是动态图），需要截图的话直接在运行得到的结果截图。

我用了matlab 工具箱里的createpde()和 solvepde()函数求解有限元，简单很多，直接从零开始太难了。
